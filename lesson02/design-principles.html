Repetition

YouTube

youtube.com

The YouTube homepage demonstrates the principle of repetition doing two things: First, youtube has a bottom navigation bar which never goes away unless you full-screen a video. This helps the user feel familiar with the page and brings a sense of "connection" between each page navigated to. Second, the header bar also keeps the same navigation and also includes some linking tags going from left to right. This styling makes the user first look left to right, then top-down when navigating the youtube browser. The style is simple but repeated enough that I could navigate this page without looking.

Visual Hierarchy

Jisho

jisho.org

This Jisho search result follows the principle of visual hierarchy by properly using item size and font weight to attract the user's vision. When I enter the website, my eye first looks at the home page's icon, then its search bar, and finally search result and definitions. This balance of weight lets me know exactly how to use the page. I know I can press the homepage's icon to return to the homepage, I know that I can type what I want to look up in the search bar, and then I realize that the queried result popped up below without needing any explicit explanation. If one were to look futher down the page, multiple related search results also appear which have a larger search result followed by a bolded definitions which also help create a visual hierarchy.

Proximity

Church of Jesus Christ

churchofjesuschrist.org

The Church of Jesus Christ's homepage uses proximity well in order to group the contents of the page together into related categories. The major links/important news are all the same size and evenly spaced on the first half of the screen. As one scrolls down, they'll notice a partition and a change in font size to show a new group for links. Lastly, the very bottom of the page, the footer, is colored differently and has another group of links which are similarly formatted. This sense of proximity allows me to recognize that this webpage is divided into sections which each serve a separate purpose. It makes navigating the page very easy and helps me remember where certain tools are when I revisit the website later.
